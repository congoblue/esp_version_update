This small app is used as a right-click function to update an ESP32 firmware version.

 Right click on the current firmware version file which is to be updated.
 
 Type the new filename in the textbox and the app will copy the "firmware.bin" file from
 
 `\.pio\build\esp32dev\firmware.bin`
 
 and will put it in the current folder with the name given in the textbox.
 
 
 The original firmware file will be deleted.
